Please find 3 files with the calibration coefficients attached.
FLAMEradioVEG_2017-08-03.csv: radiometric calibration coefficients
FLAMEradioWR_2017-08-03.csv: radiometric calibration coefficients
wl_flame.csv: wavelength calibration


How to use these vectors to calculate Radiance (L) from digital numbers (DNs) of the vegetation and the dark current contained in the data file stored on the SD card:
L(VEG) = (DN(VEG) - DC(VEG)) * FLAMEradioVEG_2017-08-03

L(WR) = (DN(WR) - DC(WR)) * FLAMEradioWR_2017-08-03

Reflectance R is calculated as:

R(VEG) = L(VEG) / L(WR)

The wavelength vector is used to plot the spectral data versus wavelengths, i.e. x-axis = wavelengths and y-axis = DNs or L or Reflectance (see attached exampled).
The unit of the wavelength vector is Nanometers.
